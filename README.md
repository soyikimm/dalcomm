Dalcom쇼핑몰 with React.js, Node.js</br>

프론트 : Html, Css, Bootstrap, React 
백엔드 : Node.js, MongoDB, Redux, Express JS</br>

개발환경 : VS Code

Github : https://github.com/soyikimm/dalcom/ </br>
Heroku Web배포 : https://pacific-crag-89307.herokuapp.com/


프로젝트 기능 :

    - 회원가입
    - 로그인, 로그아웃 구현
    - 상품 업로드 기능 구현
    - 검색기능
    - 체크박스로 카테고리 기능 구현
    - 상품 장바구니 기능 구현
    - 상품 주문 및 결제
